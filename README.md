# 3AM Club — Worth Staying Up For

A free, installable PWA that tells Malaysian / SEA football fans which 2026
tournament matches are worth losing sleep over — because every kickoff lands
between roughly midnight and 8 AM Malaysia time (MYT, UTC+8, no DST).

Pick your teams → get a night-by-night plan in MYT with a **verdict** badge
(WATCH LIVE / WORTH THE PAIN / CATCH HIGHLIGHTS / SKIP …) and a running sense of
how much sleep you're sacrificing.

## Stack
React 19 + Vite 6 · Tailwind v4 · vite-plugin-pwa · React state + localStorage.
No backend, no exposed API key — fixtures are baked at build time.

## Develop
```bash
npm install
npm run dev            # local dev server
npm test               # scoring + timezone unit tests (Vitest)
npm run build          # production build -> dist/
npm run preview        # serve the production build
```

## Fixture data
`src/data/fixtures.json` is generated by `scripts/fetch-fixtures.mjs`:

```bash
npm run fetch-fixtures
```

- **No key (default):** generates fixtures from verified public facts — the real
  12-group final draw, the 16 host venues with correct June-2026 UTC offsets, the
  real tournament date windows, and the published daily kickoff slot pattern.
  Group-stage teams are the actual draw; knockouts are TBD placeholders.
- **With `API_FOOTBALL_KEY`** (copy `.env.example` → `.env`): pulls authoritative
  fixtures from API-Football. **Verify the exact kickoff times this way before
  launch**, then re-run the script. The key lives only in this build step.

Icons: `npm run` `node scripts/gen-icons.mjs` rasterises `public/favicon.svg`.

## Deploy (Cloudflare Pages)
1. Push to GitHub.
2. Cloudflare Pages → Create project → connect the repo.
3. Build command: `npm run build` · Output directory: `dist`.
4. Deploy. The service worker + baked fixtures make it fully offline-capable.

## Tuning the scoring engine
All weights live in `src/config/`:
- `scoring.js` — stage weights, bonuses, verdict matrix.
- `teamTiers.js` — team strength buckets (seed from FIFA rankings; **adjust**).
- `rivalries.js` — marquee match-ups.

## Support
"Buy me a coffee" shows an in-app bank/e-wallet QR (no external service, no
processor fees). Replace `public/support-qr.svg` with your real QR image and tweak
`SUPPORT_NOTE` in `src/config/links.js`.
